Titulo de proyecto:
	Aplicacion de informacion de tienda de videojuegos.

Emepezando:
	El codigo se muestra con comentarios para saber que es cada vista y como reconocer lo que es.

Requisitos previos:
+PC:
 -Windosws 7/8/10, Mac OS 10.8.5, linux.
 -2gb en ram(8gb recomendado).
 -2gb de almacenamiento(4gb recomendado).
 -REsolucion minima 1280x800.
 -Java 8
 -Procesador intel 64 bits(Emulador) o dispositivo android y cable USB.
 -Conociminetos previos de programacion en Java.
 -Coocimientos basicos en XML para android.

Ejecucion:
	Descargar el proyecto y abrirlo con Android Studio.

Ejecutando pruebas:
	Utilizar emulador o dispositivo fisico para ejecutar la app.

Analisis de pruebas:
	Durante el desarrollo de esta app nos encontramos el prolema de no poder colocar toda la 
informacion en la pantalla asi que utilizamos un scroll para poder deslizar la vista
